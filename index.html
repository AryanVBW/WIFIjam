WIFIjam

Unleash the Power of Python WiFi Jamming ⚡️

WIFIjam is a modern, Python-powered WiFi jamming and deauthentication tool designed for security research, testing, and educational use.
Now with multi-OS support:

Linux Arch Mac Windows 10/11 (WSL2)

Supports 2.4GHz & 5GHz jamming, deauth attacks, and more.
Easy setup, beautiful UI, and detailed documentation for all major platforms.

Get Started →

Features

  • Jam 2.4GHz and 5GHz WiFi or Mobile Hotspot networks
  • Deauthentication Attack (kick devices off WiFi)
  • Modern, animated documentation site
  • Multi-OS support: Linux, Arch, Mac, Windows (WSL2)
  • Easy copy-paste install commands for each OS

Quick Installation

WIFIjam supports Linux (Debian/Ubuntu), Arch, Mac, and Windows 10/11 (via WSL2). For detailed, step-by-step instructions for your OS, visit the Installation page.

Linux Arch Mac Windows 10/11 (WSL2)
Tip: Use the sidebar to navigate to Installation, Usage, and more. Make sure your WiFi adapter supports monitor mode and packet injection for full functionality.

Usage

  1. Ensure you have a WiFi adapter that supports monitor mode and packet injection.
  2. Follow the installation instructions for your OS.
  3. Run the script with appropriate privileges (often sudo on Linux/Arch, or via WSL2 on Windows).
  4. Follow the on-screen prompts to select your interface and target network.

Prerequisites

  • Monitor mode and packet injection supported WiFi adapter
  • Python 3
  • Linux (Debian, Ubuntu, Fedora, Arch), Mac, or Windows (WSL2)
  • Windows 10/11 (via WSL2, see Installation page for details)
  • aircrack-ng, iw, airmon-ng, airodump-ng, aireplay-ng (see Installation page for details)
